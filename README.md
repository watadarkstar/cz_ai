# cz_ai

Generated using https://github.com/commitizen-tools/commitizen_cz_template

## Usage

Install the plugin locally:

```
pip install -e .
```

Then check that cz recognizes the plugin:

```
cz ls
```

It should appear as `cz_ai`

Then run this command to test it:

```
cz commit
```
